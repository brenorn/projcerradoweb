Trilha de Formação: Análise de Dados para a Tomada de Decisão

Desenvolva habilidades em análise de dados para apoiar a tomada de decisão em contextos de desenvolvimento sustentável.

Sobre o Curso

Este curso capacita os participantes a coletar, processar, analisar e visualizar dados para extrair insights valiosos que possam informar políticas públicas e estratégias de sustentabilidade.

Serão abordadas ferramentas como Python, R e softwares de GIS para análise de dados espaciais e não espaciais, aplicados a problemas reais do Cerrado.

Módulos do Curso

  • Introdução à Análise de Dados e Pensamento Crítico
  • Coleta e Limpeza de Dados (Web Scraping, APIs, Banco de Dados)
  • Análise Exploratória de Dados com Python e Pandas
  • Visualização de Dados com Matplotlib e Seaborn
  • Introdução ao Geoprocessamento com QGIS e GeoPandas
  • Projeto Final: Análise de um Estudo de Caso do Cerrado
